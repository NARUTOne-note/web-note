# goolgle

> 利用google 更好的查询/了解知识

## 搜索技巧

- 使用引号`""`强制进行精确匹配搜索

```
"what is javascript"
```

- `And` 运算符, 会返回和它们都相关的搜索结果

```
html AND css
```

- `OR` 操作符获取搜索词中某一个的结果

```
(javascript OR python) free course
```

- `-` 操作符将排除包含搜索词的结果

```
# 排除css
javascript -css
```

- 使用`*`通配符作为占位符，它将被任何单词或短语替换

```
how to start * in 6 months
```

- 指定网站搜索

```
site:freecodecamp.org
```

- 指定文件类型

```
filetype:pdf learn css
```

- 指定数字范围

```
ecmascript 2016..2018
```
